# go 学习笔记

## 注意

1. 变量可见性
   1. 函数内部的变量是局部变量，相当于 private
   2. 函数外部的变量是包内变量，相当于 protected
   3. 函数外部且首字母大写，全局变量，相当于 public
2. 方法返回两个结果，而你只想要一个结果 , 那另一个就用 "_" 占位，而如果用变量的话，不使用，编译器是会报错的。
3. 函数外的每个语句都必须以关键字开始（var、const、func等）
4. := 不能使用在函数外。
5. _多用于占位，表示忽略值。
